# Crypto Tracker

A minimalist, high-performance cryptocurrency dashboard built with **React**, **Zustand**, and **Tailwind CSS**. This project serves as a technical demonstration of state management, API integration, and responsive UI design.

## Key Features
* **Real-time Data:** Fetches the top 20 market cap assets directly from the CoinGecko API.
* **State Management:** Utilizes **Zustand** for lightweight, scalable global state.
* **Dynamic Search:** Instant client-side filtering of assets by name or symbol.
* **Responsive Design:** Fully mobile-responsive table layout with Tailwind CSS.
* **Error Handling:** Custom handling for API rate limits and connectivity issues.

## Tech Stack
* **Frontend:** React (Vite), TypeScript
* **State:** Zustand
* **Styling:** Tailwind CSS
* **API:** CoinGecko API
